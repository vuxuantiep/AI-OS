# 🛠️ Fähigkeiten (Skills)

Wiederverwendbare Arbeitsanleitungen für Claude und andere KI-Agenten.

## Struktur

| Ordner | Inhalt |
|--------|--------|
| `Aktiv/` | Aktuell nutzbare Fähigkeiten |
| `Archiv/` | Nicht mehr genutzte Fähigkeiten |
| `Meta/` | Metadaten und Skill-Verwaltung |
| `Vorlagen/` | Vorlage für neue Fähigkeiten |

## Nutzung

In Claude: "Nutze den Skill [Name] für [Aufgabe]"
