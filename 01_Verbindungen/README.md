# 🔗 Verbindungen

Hier werden alle Verbindungen zu externen Tools und Diensten konfiguriert.

## Struktur

| Ordner | Inhalt |
|--------|--------|
| `APIs/` | API-Konfigurationen und Secrets |
| `CLI/` | Kommandozeilen-Tools (GitHub, Google Workspace, etc.) |
| `MCP/` | MCP-Server-Konfigurationen |

## Sicherheit

API-Keys und Secrets gehören in `APIs/Geheimnisse/` (durch `.gitignore` geschützt).
