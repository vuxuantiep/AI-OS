# 📱 Lokal-Private-LLM-App

**Idee:** Mobile Gateway-App (React Native / Expo) — "souveränes ChatGPT":
On-Device-Inferenz (Gemma/Llama klein), dynamischer Inferenz-Router zum
Heim-Ollama via Tailscale/Cloudflare Tunnel, lokales RAG mit SQLite.
USP: Daten verlassen das Gerät nur, wenn der eigene Server antwortet.

**Status:** Architektur geplant (siehe `Plannung/ARCHITECTURE_PLAN.md`),
Code-Skelett in `app/`.
