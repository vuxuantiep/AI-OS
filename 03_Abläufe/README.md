# ⏱️ Abläufe (Routines)

Automatisierte, zeitgesteuerte Aufgaben.

## Struktur

| Ordner | Inhalt |
|--------|--------|
| `Lokal/` | Läuft auf diesem PC (Cron-Jobs) |
| `Remote/` | Läuft auf Server (GitHub Actions / Server-Cron) |

## Intervalle

- **Täglich/** - Einmal pro Tag
- **Wöchentlich/** - Einmal pro Woche
- **Monatlich/** - Einmal pro Monat
